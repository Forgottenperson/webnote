安裝Arch Linux

UEFI+GPT的Arch Linux安裝步驟(似乎有少寫請參考下面的連結)
P.S.請參考這個網頁的mkinitcpio

硬碟分割並建立分割區

gdisk

gdisk /dev/sda 進入分割系統環節

> o  
建立硬碟 GPT 分割表
> n
建立一個新磁區
> w
寫入系統並離開分割環節

在這裡最少須建立下列分割磁區 未提及項目則使用預設

  • Boot (開機磁區) (sda1)
    • Last sector: +512MB
    • GUID : EF00 (標記成 EFI)
  • swap (記憶體交換區) (sda2)
    • Last sector : +1G
    • GUID : 8200 (標記成 Linux swap)
  • root / (根目錄) (sda3)
    • Last sector: 皆可
    • GUID : 8300 (標記成 Linux 硬體空間)

mkfs.vfat -F32 /dev/sda1

將 Boot 磁區格式化成 FAT32

mkfs.ext4 /dev/sda3

將 root 磁區格式化成 ext4

mkswap /dev/sda2 swapon /dev/sda2

將 swap 磁區標記成 swap

掛載硬碟

mount /dev/sda3 /mnt

mkdir /mnt/boot 建立 boot 資料夾

mount /dev/sda1 /mnt/boot

/etc/pacman.d/mirrorlist 是 pacman 套件管理的下載來源

開始安裝主系統

pacstrap -i /mnt base base-devel linux

建立 fstab (開機自動掛載項目)

genfstab -U -p /mnt >> /mnt/etc/fstab

arch-chroot /mnt /bin/bash

移動至安裝完的系統中

修改語系

修改 /etc/locale.gen

把會使用到的語系取消註解後儲存

執行 locale-gen

echo LANG=en_US.UTF-8 > /etc/locale.conf export LANG=en_US.UTF-8

修改時區

ln -s /usr/share/zoneinfo/Asia/Tokyo /etc/localtime

hwclock -u -w

修改 hostname

echo 名稱 > /etc/hostname

修改 root 密碼

passwd

建立開機映像檔

mkinitcpio -p linux

載入模組

modprobe dm-mod

安裝 grub 開機管理清單

pacman -S grub dosfstools efibootmgr
grub-install --target=x86_64-efi --efi-directory=/boot --bootloader-id=arch_grub --recheck --debug
grub-mkconfig -o /boot/grub/grub.cfg

mkdir /boot/EFI/boot
cp /boot/EFI/arch_grub/grubx64.efi  /boot/EFI/boot/bootx64.efi
